# Jojo Valentine Website

A romantic, interactive single-page website built for Jojo with a heart‑puzzle intro, photo gallery, love notes, and a tulip animation reveal.

## Features
- Heart puzzle gate (drag to fit) that unlocks the site
- Smooth scrolling and subtle motion (with reduced‑motion support)
- Responsive layout for desktop, tablet, and mobile
- Curated gallery layout with safe cropping
- Animated tulip “Yes” moment with a gratitude message

## How to run locally
Open `index.html` in your browser.

## Deploy with GitHub Pages
1. Push this repo to GitHub.
2. In the GitHub repo: **Settings → Pages → Build and deployment**
3. Set **Source** to `Deploy from a branch`.
4. Select branch `main` and folder `/root`.
5. Save. Your site will be live at the URL GitHub shows.
